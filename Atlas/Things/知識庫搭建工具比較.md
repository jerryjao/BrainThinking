# 知識庫怎麼搭？

整理：
- FastGPT：適合快速搭企業知識庫平台，工作流、問答、資料集管理完整。
- LLM Wiki：適合個人桌面資料管理、本地查閱與文件歸檔。
- llm-wiki-agent：適合讓 Claude / Codex / Gemini 自動讀、整理、維護個人知識庫。
- OpenKB：適合先搭輕量開源底座，做最小可用版本。

選擇建議：
- 想快速搭完整知識庫平台，先看 FastGPT。
- 偏個人桌面資料管理，看 LLM Wiki。
- 想讓 agent 自動維護知識庫，看 llm-wiki-agent。
- 想先搭個輕量開源底座，再看 OpenKB。
