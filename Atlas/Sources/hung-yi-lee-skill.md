# voidful/hung-yi-lee-skill 整理摘要

## 內容整理
- SKILL.md：主要入口，定義這個 skill 的使用方式與行為。
- AGENTS.md：維護規範與 wiki/schema 說明。
- scripts/hungyi_kb.py、scripts/hungyi_graph.py：知識庫同步、搜尋、編譯、圖譜分析與檢查工具。
- raw/youtube/：YouTube 影片 metadata 與逐字稿快取。
- wiki/：知識庫內容，包含入口、主題地圖、查詢流程、圖譜、topics、series、video catalog。
- references/：persona、spirit、work、sources 等參考文件與 examples。
- outputs/query-briefs/：查詢摘要輸出。
- README.md / README.en.md：專案總覽與使用說明。

## 重點
- 這個 repo 不是單純的語錄集合，而是把李宏毅老師的課程內容整理成可查詢的知識庫與知識圖譜。
- README 提到來源包含 478 支 YouTube 影片、27 份完整逐字稿、8 個主題頁面與 4 份精選研究參考資料。
- 已建成 916 節點的知識圖譜，並提供 CLI 工具與 wiki 內容支援搜尋與導覽。
