內容展開：
這則 Threads 在介紹一個由 Netflix 工程師開源的工具，重點在解決 AI Agent 上下文膨脹與 token 成本過高的問題。

核心意思是：
- AI Agent 雖然越來越強，但每次工作都會累積更多上下文。
- 上下文越多，token 消耗越快，成本與效率都會受影響。
- 在 Claude Code、Cursor、Codex 這類工具中，AI 會持續讀檔、看 log、抓 tool output、掃 codebase、吃 RAG chunks，因此更容易發生上下文失控。

原文摘錄：
「一個 Netflix 工程師開源了超狂工具。它要解決的問題很簡單：AI Agent 越來越好用，但上下文越塞越多，token 也越燒越兇。尤其你在用 Claude Code、Cursor、Codex 這類工具時，AI 會讀檔案、看 log、抓 tool output、掃 codebase、吃 RAG chunks。」
